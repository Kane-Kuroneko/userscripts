# 更新日志 (7.0.0 - 当前版本)

## 🎉 主要功能更新

### 🌐 新增 fzgamer.com 站点支持
- **搜索按钮集成**：在 fzgamer.com 站点添加"去Steam搜索"按钮，方便用户快速跳转至Steam平台搜索游戏
- **模态弹窗功能**：支持点击文章列表项弹出模态窗口浏览内容，提供更流畅的浏览体验
- **页面布局优化**：
  - 将游戏视频标题、视频框和温馨提示框移动到版本介绍上方
  - 优化页面元素排列顺序，提升视觉层次
  - 简化侧栏展示，移除"近期游戏热榜"以上的冗余内容

### 🔧 fzgamer.com 弹窗功能修复
- **修复 main body 区域弹窗失效问题**：
  - 问题：原选择器仅匹配 `.widget-ajaxpager`，导致 main body 区域的文章卡片无法触发弹窗
  - 解决：扩展选择器为 `.widget-ajaxpager, .ajaxpager`，兼容不同页面结构的容器
  - 影响：现在首页 main body 区域、侧边栏等所有位置的文章卡片均可正常打开弹窗
- **优化事件监听逻辑**：移除调试代码，避免在点击非文章区域时产生 TypeError 错误


### 💡 弹窗浏览体验升级  
- 对gamer520的其它位置的卡片也适配了弹窗浏览功能
